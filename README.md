# DeliveryApp

Sistema de delivery com chat AI para pedidos.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Funcionalidades

- Chat AI para pedidos de clientes
- Dashboard Kanban para gestão de pedidos
- Gestão de cardápio (categorias e produtos)
- Configurações do estabelecimento
- Autenticação com Supabase
- Rate limiting com Redis

## Technologies

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- shadcn/ui
- Supabase
- OpenRouter (AI)
